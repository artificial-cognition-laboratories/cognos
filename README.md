# cognos
A cognitive operating system for intelligent agents. Foundation of Axon.

Cognos is a private cognitive operating system developed by Arc Labs. It serves as the foundation for intelligent agents, including **Axon**.

This repository contains high-level documentation about the architecture, philosophy, and design principles of Cognos. The implementation is proprietary and not included here.

## Overview

- Event-sourced cognitive runtime
- Perception → Cognition → Action loops
- Capsules as mediated embodiments for controlled interactions
- Designed for observability, replay, and debugging

## Relationship to Axon

Axon is a product built on top of Cognos. Cognos provides the underlying substrate for agent intelligence, while Axon exposes developer-facing APIs and tools.
